# sin()

Returns the sine value of the specified number.

## Syntax

`sin(`*number*`)`

## Parameters

| Name | Type | Required | Description |
|--|--|--|--|
| *number* | real | &check; | The number for which to calculate the sine.|

## Returns

The sine of *number*.
