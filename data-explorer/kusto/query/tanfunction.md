# tan()

Returns the tangent function.

## Syntax

`tan(`*x*`)`

## Parameters

| Name | Type | Required | Description |
|--|--|--|--|
| *x* | real | &check; | The number for which to calculate the tangent. |

## Returns

The result of `tan(`*x*`)`
